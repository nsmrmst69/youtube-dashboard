「依頼する」→「依頼中」：GASが担当編集者のChatworkへ自動で依頼メッセージ送信 ＋ 編集者管理シートを自動更新
「先方確認中」：GASがクライアントのChatworkへ自動で確認依頼メッセージ送信

「済」：GASが編集者管理シートの該当スロットを「空き」に自動更新
04 — Chatwork チャンネル構成

💼 チャンネル1 — クライアント

クライアントとの全コミュニケーション
  • 現:手動 素材URL・編集指示の受領 → Sheetsへコピペ
  • →自動化 受信メッセージをWebhookで検知 → Sheetsへ自動書き込み
  • →自動化 確認依頼メッセージを自動送信（ステータス「先方確認中」トリガー）
  • →自動化 修正やりとり（部分自動化）

✂️ チャンネルn — 外注編集者（n人分）

編集者ごとに独立したチャンネル
  • →自動化 編集依頼メッセージを自動送信（ステータス「依頼する」トリガー）
  • →自動化 依頼内容（動画タイトル・素材URL・編集指示）を自動添付
  • 現:手動 修正やりとり
  • 現:手動 納品連絡の受領

👤 チャンネル2 — 同僚

公開前ダブルチェック用
  • →自動化 M列「公開設定」✅でダブルチェック依頼を自動送信
  • →自動化 確認完了返信でステータスを「済」に自動更新（Webhook）
05 — トリガー別 自動化フロー
1

素材受領の自動記録

クライアントからの素材URLと編集指示をSheetsへ自動書き込み
💼
クライアント（Chatwork チャンネル1）
メッセージを送信
・ギガファイル便URL
・編集指示テキスト
⚙️
GAS（Chatwork Webhook → 解析）
受信メッセージを自動解析してURL・指示を抽出
📊
Googleスプレッドシート（D列 メモ）
受信日をキーに該当行のD列へ
素材URL ＋ 編集指示を自動書き込み
2

編集依頼の自動送信

ステータスを「依頼する」に変えるだけで編集者へ自動依頼
⚡
トリガー（K列 → 「依頼する」）
F列から担当編集者を特定し処理開始
⚙️
GAS（自動処理）
依頼メッセージを自動生成
・動画タイトル（D列）
・素材URL（D列メモから取得）
・編集指示（D列メモから取得）
・納品予定日（I列）
✂️
外注編集者（Chatwork チャンネルn）
担当編集者のチャンネルへ自動送信
📊
Sheets 自動更新
K列「依頼中」に自動更新
編集者管理シートの該当スロットも更新
3

クライアント確認依頼の自動送信

ステータスを「先方確認中」に変えるだけで確認依頼が飛ぶ
⚡
トリガー（K列 → 「先方確認中」）
GASが即座に処理開始
⚙️
GAS（確認依頼メッセージ生成）
定型文に動画情報を自動埋め込み
・動画タイトル
・完成動画リンク（L列）
・確認期限（自動計算）
💼
クライアント（Chatwork チャンネル1）
確認依頼メッセージを自動受信
4

YouTube公開後ダブルチェック依頼

公開設定チェックで同僚へ自動依頼、返信で「済」に自動更新
⚡
トリガー（M列「公開設定」✅チェック）
チェックボックスONでGASが起動
👤
同僚（Chatwork チャンネル2）
ダブルチェック依頼を自動受信
・動画タイトル
・YouTube公開URL
・チェック項目リスト
📊
確認完了後（Sheets 自動更新）
K列を「済」に自動更新
編集者管理スロットを「空き」に自動更新
5

オンラインサロン 公開完了報告の自動送信

W列にチェックを入れるだけで、クライアントへの報告が完全自動化（別シートの定型文テンプレートが不要に）

手動コピペ

💼
クライアント
報告メッセージを受信
自動化後のフロー

チェックONでGASが即座に起動
⚙️
GAS（報告メッセージ自動生成）
同じ行から必要データを自動取得
・動画タイトル（S列）
・投稿ページURL（T列）
・Vimeo_URL（U列）

💼
